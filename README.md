# Quasar App

> A Quasar project

## Build Setup

``` bash
# install dependencies
$ npm install

# serve with hot reload at localhost:8080
$ quasar dev

# build for production with minification
$ quasar build

# lint code
$ quasar lint
```

``` 
About filename strategy

Component files are named with StudlyCase
Script files with function as constructor are namen with StudlyCase
Script files without function as constructor (return array, function, object) are namen with cameCase
```
